## ğŸ“Œ Project Overview

The robot was designed and built as a small-scale academic prototype to explore multi-legged locomotion and system integration under real constraints.

The main goal of this project was not to develop a commercial platform, but to understand and validate the complete development cycle of a robotic system, from design to testing.

Key objectives included:

- Designing and manufacturing custom mechanical parts using 3D printing  
- Integrating servomotors, sensors, and control electronics  
- Implementing basic gait and heading control  
- Validating behavior through simulation and physical experiments  
- Producing structured technical and user documentation
